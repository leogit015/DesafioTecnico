# Desafio Técnico

Este é o repositório do **Desafio Técnico**, um projeto desenvolvido em Java que utiliza o Maven para gerenciamento de dependências e build.

## Estrutura do Projeto

O projeto contém os seguintes diretórios e arquivos:

- **.gitignore**: Arquivo que define os arquivos a serem ignorados pelo Git.
- **.mvn/**: Pasta contendo arquivos relacionados ao wrapper do Maven.
- **pom.xml**: Arquivo de configuração do Maven que gerencia dependências e plugins.
- **src/**: Contém o código-fonte do projeto.

### Diretório `src/main/java`

Este diretório contém o código-fonte principal do projeto, incluindo as classes de modelo e a classe principal:

- **com/iniflex/teste/Principal.java**: Classe principal que executa a lógica principal do programa.
- **model/Funcionario.java**: Modelo de dados para a classe `Funcionario`.
- **model/Pessoa.java**: Modelo de dados para a classe `Pessoa`.

### Arquivos de Configuração

- **.editorconfig**: Arquivo de configuração para padronizar a formatação de código em diferentes editores de texto.
- **.gitattributes**: Configurações relacionadas ao Git, como atributos de arquivos.

## Como Rodar o Projeto

### Requisitos

- Java 11 ou superior
- Maven 3.6 ou superior
